This ZIP file contains the complete contents of a collection.

INCLUDED FILES
==============
- *.html - HTML version of the collection.  This can be viewed by opening the bk01-toc.html file in your browser.
- cover.html amd cover.png - cover page and image from EPUB
- biXX.html - Bibliography files
- index.html - title page from EPUB
- collection.xml - XML file that defines the structure of the collection.  It cannot be reimported in the editing interface.
- content.css - CSS file for styling the HTML files.
- *.dbk - Docbook files used to generate the offline HTML/EPUB files.

INCLUDED DIRECTORIES
==================== 
- mXXXXX (e.g. m10000) - 
  - index.cnxml - Contains the cnxml (XML source) files for each module 
  - index_auto_upgrade.cnxml - CNXML 0.7 version of module with enhanced metadata
  - any images or multimedia files that are part of the module.
